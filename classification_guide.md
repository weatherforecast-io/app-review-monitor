# 앱스토어 리뷰 분류 가이드

이 문서는 Claude AI가 리뷰를 분류할 때 참고하는 기준입니다.
이 파일을 수정하면 분류 방식이 바뀝니다.

---

## 중요도 (Importance)

리뷰의 긴급성과 비즈니스 영향도를 기준으로 4단계로 분류합니다.

### 🚨 critical
즉시 대응이 필요한 심각한 문제. 사용자가 앱을 정상적으로 사용할 수 없는 상태.

- 앱이 실행되지 않거나 크래시가 반복됨
- 데이터가 손실되거나 삭제됨
- 결제했는데 서비스가 제공되지 않음
- 개인정보 유출 또는 보안 관련 문제
- 업데이트 후 기존 기능이 완전히 작동 안 함

### ⚠️ high
빠른 확인이 필요한 문제. 핵심 기능에 영향을 주지만 앱 자체는 사용 가능.

- 특정 기능에서 버그 발생
- 앱이 자주 멈추거나 매우 느려짐
- 로그인/계정 관련 문제
- 알림이 오지 않는 등 주요 기능 오작동
- 배터리 과다 소모

### 📌 medium
개선하면 좋은 사항. 기능은 작동하지만 사용자 경험에 불만.

- UI/UX 불편 사항
- 원하는 기능이 없다는 요청
- 다른 앱 대비 아쉬운 점
- 광고 관련 불만
- 별점 3점대의 애매한 피드백

### ✅ low
긍정적 리뷰 또는 사소한 피드백. 즉각 대응 불필요.

- 만족 표현, 칭찬, 추천
- 사소한 디자인 의견
- 이미 알려진 사항이거나 의도된 동작에 대한 코멘트

---

## 카테고리 (Category)

리뷰의 주요 내용을 기준으로 아래 카테고리 중 **가장 적합한 1개**를 선택합니다.

### stability (안정성)
앱 크래시, 강제 종료, 앱이 열리지 않음, 화면 멈춤, 무한 로딩

### performance (성능)
속도 느림, 렉/지연, 배터리 소모, 메모리/저장공간, 발열

### ux (사용성)
UI 불편, 찾기 어려운 기능, 복잡한 인터페이스, 디자인 불만, 접근성

### feature_request (기능 요청)
새 기능 요청, 다른 앱에 있는 기능 원함, 기존 기능 개선 요청

### data_accuracy (데이터 정확도)
날씨 예보 정확도, 미세먼지 수치 오차, 위치 기반 정보 오류, 데이터 불일치

### notification (알림)
알림이 안 옴, 알림이 너무 많음, 알림 시간 부정확, 알림 커스터마이징

### login_account (로그인/계정)
로그인 실패, 계정 연동 문제, 비밀번호 관련, 소셜 로그인 오류

### billing (결제/구독)
구독 관련, 결제 오류, 환불 요청, 가격 불만, 무료/유료 전환

### widget (위젯)
홈화면 위젯 관련, 위젯 업데이트 안 됨, 위젯 디자인, 위젯 크기

### localization (현지화)
번역 오류, 언어 지원, 지역별 날씨 서비스, 단위(온도/거리) 설정

### ads (광고)
광고 과다, 광고 위치 불만, 광고 제거 요청

### positive (긍정)
만족, 칭찬, 추천, 감사 표현 (별점 4-5점이면서 불만이 없는 경우)

### other (기타)
위 카테고리에 해당하지 않는 경우

## 분류 시 참고사항

- 하나의 리뷰에 여러 주제가 섞여있으면, **가장 핵심적인 내용** 기준으로 분류
- 별점이 낮아도(1-2점) 내용이 긍정적이면 내용 기준으로 분류
- 별점이 높아도(4-5점) 불만이 포함되어 있으면 불만 내용 기준으로 분류
- 중요도 판단 시 별점도 참고하되, **리뷰 내용**이 더 중요한 판단 기준
